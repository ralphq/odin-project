Pull

Description

A "pull" workout will target the muscles used to pull things towards your body, such as the back, biceps, and certain muscles in the shoulders.

Exercises

  1. Lat Pulldowns
  2. Cable Rows
  3. Bicep Curls

Tips

  • Each of these exercises should be done for 3 sets
  • Each set should be in the 8-12 rep range
  • Be sure to avoid "body english" in order to maximize gains
