Push

Description

A 'push' workout generally targets the muscles used when pushing things away from your body, such as the triceps, chest, and shoulders.

Exercises

  1. Barbell Bench Press
  2. Dumbbell Lateral Raises
  3. Cable Chest Flies

Tips

  • Each of these exercises should be done for 3 sets
  • Each set should be in the 8-12 rep range
  • Be sure to avoid "body english" in order to maximize gains
