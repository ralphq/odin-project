Legs

Description

A comprehensive leg workout will target various muscles including the quadriceps, hamstrings, glutes, and calves.

Exercises

  1. Squats
  2. Deadlifts
  3. Calf Raises

Tips

  • Each of these exercises should be done for 3 sets
  • Each set should be in the 8-12 rep range
  • Be sure to avoid "body english" in order to maximize gains
